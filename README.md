## Run Locally

Clone the project

```bash
  git clone https://github.com/jishant-AKA-jk/Happening.git
```

Go to the project directory

```bash
  cd Happening
```

Install dependencies

```bash
  npm install
```

Run App

```bash
  #For Android
  npx react-native run-Android


  #For iOS
  cd ios
  pod install
  cd ..
  npx react-native run-ios
```
